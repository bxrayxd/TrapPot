  \section*{Abstract}
\end{center}

\addcontentsline{toc}{section}{Abstract}

\begin{spacing}{1.5}
As Saudi Arabia expands its Internet of Things (IoT) infrastructure under Vision 2030, the proliferation of insecure devices has increased susceptibility to large-scale botnet attacks. This project introduces TrapPot, an integrated system using Artificial Intelligence (AI) to observe, analyze, and automatically classify malicious activities targeting IoT environments to enhance proactive security readiness
The research employs a quantitative case study methodology, utilizing a medium-interaction Cowrie honeypot to emulate Telnet and Secure Shell (SSH) services within a strictly isolated network
. Real-world attack logs are processed through the ELK Stack (Elasticsearch, Logstash, and Kibana) for real-time monitoring, while a data pipeline extracts behavioral features, such as login attempts and session duration, to train a Random Forest (RF) classification model
The design phase has produced a comprehensive system architecture and feature engineering strategy focused on identifying four distinct attack types: brute-force, scanning, malware download, and command execution
. This methodology is engineered to achieve a target classification accuracy of 96%, ensuring high confidence in the identification of harmful activities
TrapPot distinguishes itself by combining automated machine learning with real-time visualization tools to address specific regional cybersecurity workforce development needs in Saudi Arabia
. The successful completion of this project provides a practical framework for transitioning from reactive to proactive defense strategies within the rapidly evolving IoT threat landscape.\\

  {\bf Keywords:} TrapPot, Internet of Things (IoT), Honeypot, Random Forest (RF), Cybersecurity, Artificial Intelligence (AI).
\end{spacing}
